# Boring Avatars

## Install

```
yarn add import boring-avatars
```

or

```
npm install import boring-avatars
```

## Usage

```jsx
import Avatar from "boring-avatars";

<div>
  <Avatar size="64px" variant="abstract" name="Mike" />
</div>
```

### Props

| Prop    | Type                           |
| ------- | ------------------------------ |
| size    | number or string               |
| name    | string                         |
| variant | oneOf: "abstract", "geometric" |
